# 🚗 Car Rental App: High-Performance Car Rental Engine
FastAPI + React + Stripe | AI-Augmented Full-Stack Architecture

## 📌 Project Overview
DriveTime is an enterprise-grade car rental platform designed to demonstrate the power of Asynchronous Python and Modern Component-Based UI. This project bridges the gap between simple CRUD apps and high-stakes financial applications by implementing real-time availability logic and secure payment orchestration.

## 🚀 Core Innovation: The 2026 Tech Stack
This repository serves as the blueprint for the Udemy Innovation Grant 2026 curriculum, focusing on:

- AI-Orchestrated Development: Utilizing Cursor and GitHub Copilot for high-velocity scaffolding and unit testing.

- Atomic Transactions: Preventing double-booking through advanced SQLAlchemy async sessions.

- Stripe Webhook Listener: A robust, cryptographically verified bridge for asynchronous payment confirmation.

- Universal Dev Environments: Full containerization via Docker Compose for 1:1 parity across Windows, Mac, and Linux.

## 🛠 Technical Architecture
### Backend (FastAPI)

- Async/Await: Leveraging Python’s asyncio for non-blocking I/O.

- Pydantic V2: Strict data validation and settings management.

- JWT & RBAC: Secure Authentication and Role-Based Access Control.

- Pytest: 100% coverage strategy for financial logic.

### Frontend (React 19)

- Native CSS: Responsive, utility-first design for mobile-first rental browsing.

- TanStack Query: Efficient server-state management and caching.

- Stripe Elements: Secure, PCI-compliant payment UI integration.

### Cloud & DevOps (AWS)

- CI/CD: Automated testing and deployment via GitHub Actions.

- Terraform/IAC: (Planned) Infrastructure as Code for AWS resource provisioning.

- AWS ECS/Fargate: Containerized deployment for high scalability.

## 🏗 Installation (The "One-Click" Setup)
Ensure you have Docker installed, then run:

```Bash
docker-compose up --build
```

The API will be available at localhost:8000/docs and the Frontend at localhost:3000.
